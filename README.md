# Face_Swap_Pipeline
An automated pipeline that takes 2 face images, multiple scenes, generates a prompt via Google Gemini, and then creates a final image using Seedream 4.0./Автоматический конвейер, который берёт 2 изображения лица, много сцен, генерирует промпт через Google Gemini, после чего создаёт финальное изображение с помощью Seedream 4.0.
